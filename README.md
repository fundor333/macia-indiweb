# Macia WebMention

Hugo Module for adding the header for the WebMention

~~~ html

~~~

## Configs

The default config for params is this

~~~ yaml

  pingback: 
  micropub: 
  microsub: 
  authorization_endpoint: 
  token_endpoint: 
~~~
